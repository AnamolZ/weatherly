### Weather App 
> Experience the weather, beautifully.

This modern Weather App offers a seamless and visually stunning way to check real-time weather conditions for any city in the world. Built with a focus on aesthetics and user experience, the application features a premium glassmorphism design that layers translucent elements over a vibrant background, creating a sense of depth and modern elegance. Users can instantly retrieve accurate data including temperature, humidity, and wind speed, powered by the reliable OpenWeatherMap API.
